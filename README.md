# SuperLoop
无线列表模块
在本案例中，我只展示最常用的其中一个，满足大部分游戏开发（物品、装备背包，卡牌背包）等常用功能
具体模块请去个人主页PlugIns 去下无线列表插件根据需求自行学习，和使用
###***ScrollRect.verticalScrollbarVisibility cannot be set to AutoHideAndExpandViewportUnityEngine.Debug:LogError (object)SuperScrollView.LoopGridView:InitGridView 
如果碰到以上错误
解决方法
1.选中 LoopGridView 对象，2.找到 ScrollRect 组件  3.将 Vertical Scrollbar Visibility 改为 AutoHide 或 Permanent 4.不要用 AutoHideAndExpandViewport
